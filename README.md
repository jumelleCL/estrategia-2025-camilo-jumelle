# Estrategia 2025

## Based turn

Escogi hacer un juego de turnos
El juego consiste en por turnos vas elijiendo si moverte o atacar, solo podes hacer un movimiento por turno, hasta terminar con todos tus personajes muertos o todos los enemigos muertos.

## Controles del juego

1. Click izquierdo -> Movimiento personajes (drag and drop) 
2. Click derecho -> HUD informacion de ataques/movimientos


## Requisitos Basicos

- [x] Capacitat d’elecció, per exemple diferents torretes.
- [x] El HUD es crea i s’emplena per AngelScript.
- [ ] Ús d’esdeveniments i dispatchers, tot ha d’estar desacoblat.
- [x] Enemics o objectius a on atacar i en fer-ho que morin.
- [x] Ús d’un actor component o Interfície.


## Requisitos opcionales



- [ ] Fer una pool d’objectes.
- [ ] Augment progressiu de dificultat.
- [ ] Mapa de joc generat de manera procedimental.
- [x] Menú inicial amb diversos nivells.
- [ ] “Waves” personalitzables a on posar quantitat d’enemics i tipus d’enemics a aparèixer.
